# f(n) = 1+2+3+....+n ==>  iterative vs recursive

- iterative
```python
def f(n):
   result = 0; 
   for i in range(n+1):
        result += i
   return result     

f(5)
# 1+2+3+4+5 = 15
```
- recursive:a defined function can call itself
  - f(n) = 1+2+3+...+(n-1)+n
  -      = f(n-1) + n
  - stopping conditopn:f(1) =1     
```python
def f(n):
   if n ==1:
     return 1
   else:
     return f(n-1) + n

f(5)
```
